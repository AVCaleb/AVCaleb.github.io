港口水工建筑物

码头概论

码头分类

重力式——地质条件需要比较好

板桩式——依靠侧向土抗力（板桩入土部分左右的土抗力）还有锚碇结构。

优点：耐久性好（相对高桩码头，没有重力式好），结构简单（相对重力式码头），材料用量少（相对重力式），便于预制（可以缩减工期）。可以先打桩，后开挖港池（因为之后要做基础）

缺点：波浪反射严重（对直立部分有一个长期的推力。且部分水流的冲掏作用，会不断侵蚀基床），泊稳条件差，对钢板桩需采用防锈措施，增加费用，对开挖超深反应敏感（因为会改变侧向土抗力）

适用条件：能打板桩的地基（相对的，不能过于浪费人力物力），万吨级以下的泊位（其对于万吨级的船水平向难以支撑），适用于有掩护的海港（波浪反射严重，以及波浪淘刷作用）

透空式（高桩）

工作原理：通过桩台将作用在码头上的荷载经桩基传给地基

优点：波浪反射小（将危害转移掉），泊稳条件好，砂石用量少，对开挖超深适应能力强

缺点：对地面超载、工艺变化的适应能力差（主要受力：摩擦力、应力。桩基抗弯、抗剪能力差，一旦超载，就会弯矩过大，码头失效）水平承载能力低。耐久性差，须设叉桩（大直径管柱例外）（然而无法改善与地基的弱作用力，无法改变本质上的受力特征）

适用条件：软土地基（重力式肯定不行，因为要地基好，软土不适合。板桩码头，需要侧向土抗力，然而软土的侧向土抗力小，所以也不适用。只能用高桩码头，因为其桩基可以容易打到深的地方。），不适用于大吨位。

混合式

因地制宜，组合使用。

码头组成部分

  1. 上部结构的作用
    1. 承上启下
    2. 设置码头设施
  2. 下部结构的作用
    1. 支撑上部结构
    2. 传递荷载
  3. 码头设别——船舶系泊、装卸作用

码头结构上的作用及组合

结构上的作用——施加在结构上的外力（直接作用（集中力和分布力，即荷载）），以及引起结构外加变形和约束变形的原因（间接作用（地基沉降、砼收缩、温度变形））

  1. 作用的分类

    1. 按时间变异分类
      1. 永久作用：在设计基准期内（设计时所确定的工程项目的寿命。工程界分为好几档。港口工程钢筋混凝土的设计基准期为50年。），其量值随时间的变化与平均值相比可忽略不计的作用，如自重力、预加应力，土重力，永久作用引起的土压力
      2. 可变作用：在设计基准期内，其量值随时间的变化与平均值相比不可忽略不计的作用。如堆货、流动起重运输机械、可变作用引起的土压力、船舶荷载、波浪力。
      3. 偶然作用：在设计基准期内，不一定出现，但一旦出现其量值很大且持续时间很短的作用，如地震作用
    2. 按空间位置的变化分类
      1. 固定作用：在结构上具有固定分布的作用，如自重力
      2. 自由作用：在结构的一定范围内可以任意分布的作用，如堆货
    3. 按结构的反应分类（看有没有冲击力）
      1. 静态作用：加载过程中产生的加速度可以忽略不计
      2. 动态作用：加载过程中产生的加速度不可以忽略不计，如船舶的撞击力、汽车荷载

  2. 作用组合和作用代表值的取值

    1. 组合原则

      1. 对实际有可能同时出现的作用，应按其最不利情况进行组合
      2. 对于不同的计算项目，应分别按其最不利情况进行组合
      3. 受水位影响的水工建筑物，应把水位作为一个组合条件

    2. 作用组合和作用代表值的取值

      1. 作用组合

        港口工程结构上同时出现的作用应分别按承载能力极限状态和正常使用极限状态考虑作用效应的组合。一旦超出承载能力极限状态，结构和构建丧失承载能力；超出正常使用极限状态时，结构和构建不能满足适用性和耐久性的要求（例如墙出现了很大的裂缝，但是不会倒）。

        1. 承载能力极限状态验算
          1. 持久作用：永久作用和持续时间比较长的可变作用组成的作用效应的组合
          2. 短暂组合：包含持续时间较短的可变作用组成的作用效应的组合
          3. 偶然组合：包含偶然作用组成的作用效应的组合
        2. 对正常使用极限状态的验算：持久状况（短期效应（频遇）、长期效应（准永久））和短暂状况

      2. 作用代表值的取值

        1. 作用代表值的分类——标准值1、频遇值2、准永久值3
        2. 不同作用的代表值
          1. 永久作用——1
          2. 可变作用——123
          3. 偶然作用——要根据资料或工程经验综合分析
        3. 对承载能力极限状态可变作用
          1. 持久作用：主导可变作用1，非主导可变作用取组合值（组合值是将标准值乘以组合系数，\(\psi=0.7\)）
          2. 短暂作用：查规范
          3. 偶然作用：查规范
        4. 对正常使用极限状态可变作用
          1. 持久状况的短期效应组合：取可变作用的频遇值（组合值是将标准值乘以组合系数，\(\psi=0.8\)）
          2. 持久状况的长期效应组合：取可变作用的准永久值（组合值是将标准值乘以组合系数，\(\psi=0.6\)）
          3. 短期状况：标准值

    3. 设计表达式

      1. 承载能力极限状态 \[ S_d\leq R_d \] 其中，\(S_d\)为作用效应值，\(R_d\)为抗力值

        组合系数（概率）、分项系数（可靠度）、结构重要性系数（即安全系数）

      2. 正常使用极限状态 \[ S\leq R \] S——作用效应设计值

        R——限值，如规定的最大变形，使功能正常是使用的一个值

    4. 说明

      1. 正常条件下，结构使用过程中的状况为持久状况
      2. 结构施工和安装等持续时间较短的状况为短暂状况
      3. 结构承受设防地震等持续时间很短的状况为偶然状况

码头地面组成荷载

  1. 堆荷荷载与人群荷载

    1. 堆荷荷载
      1. 考虑因素
        1. 装卸及码头堆存工艺
        2. 货种和包装方式
        3. 货物批量和堆存期：小批，临时，小堆，利于货物的运转；大批，堆存期长，大堆，提高库场利用率。
        4. 码头结构型式：不同码头对荷载敏感程度不同
        5. 管理水平：管理严格，则利用率高，不会出现超载
      2. 堆荷的分布与取值——码头前沿、前方堆场、后方堆场
        1. 前沿地带——一般不堆货
          1. 多数情况下不堆货，主要作为流动起重云顺机械的通道
          2. 少数情况，货物临时堆放
          3. 前沿地带考虑堆货值是从安全出发，并非从堆货出发
          4. 宽度：有门机，14M；无门机10m（海港），4~8m（河港）
          5. 荷载值\(q_1\)，一般取20kPa（主要基于起重机等设备）
        2. 前方堆场——效率最高
          1. 宽度：有门机18(23)m，对应门机最大吊幅25(30)m；无门机，取仓库宽度
        3. 后方堆场：前方堆场以后的堆场。
          1. 堆存批量较大，堆存期较长，堆存较重
          2. 宽度主要取决于规划
          3. 取值对码头结构无多大影响
    2. 人群荷载（略）——作人行引桥、浮桥设计时需要考虑

  2. 流动起重运输机械荷载

    起重运输机械：门机、起重机、岸桥、缆车

    装卸搬运机械：叉车、电瓶搬运车、牵引车、单斗车

    1. 门机

      1. 表示方法：\(M_h-Q-L\)，其中，Q为吊重，L为吊幅
      2. 单机作业主要考虑三种工作状态（三种吊臂位置，最大吊幅，最小吊重；最小吊幅，最大吊重）
      3. 两台门机一般只考虑工作状态1
      4. 不考虑门机荷载的冲击系数（比如起吊、卸货时，但是不考虑，因为风压已经考虑了这一部分内容）
      5. 门机荷载作用下，计算土压力时，应将门机荷载换算为等代线荷载（把门机荷载均匀分布处理）

    2. 轮胎式起重机

      1. 主要用于中小码头，无门机情况
      2. 码头设计时，对于轮胎式只考虑两种情况：最大吊幅，最小吊重；最小吊幅，最大吊重
      3. 当采用最大吊重时，一般不考虑冲击系数（因为一般起重的货物较小，工人娴熟度较高），否则应考虑1.1~1.3的冲击系数

    3. 缆车荷载（斜坡码头上、下坡主要运载工具）

      轮压计算： \[ P=g(Q+G)K_t/n \ \ \ \ (KN) \] 式中：\(Q，G\)——载重和自重（t），\(n\)——轮数，\(K_t\)——轮压不均匀系数

    4. 岸桥荷载——根据具体机型确定其荷载

  3. 铁路荷载（现在很少使用）（对板桩码头、高桩码头影响很大）

    1. 进港机车有：
      1. 按使用性质分：干线机车、调车机车
      2. 按动力型式分：蒸汽机车、内燃机车、电力机车（现在可以多考虑）
    2. 进港车辆有：
      1. 普通车辆：载重量60吨以下
      2. 特种车辆：载重量60吨以上
      3. 荷载取值
        1. 用“中-活载”（轴压）取代轮压，直接计算轮压时除2。铁路机车在码头上行驶一般不考虑冲击力、离心力、制动力（行驶速度慢的缘故）
        2. 对直接承受荷载的结构和构件（如梁、单向板、轨枕），“中-活载”要乘以荷载系数K。

  4. 汽车荷载

    1. 分级取值：五个等级
    2. 冲击系数：1.1~1.3

  5. 船舶荷载

    1. 按作用方式可分为：系缆力，挤靠力，撞击力

    2. 船舶系缆力（由风、浪、水流、冰产生）

      1. 其中，纵向是平行码头岸线的方向，横向是垂直码头岸线的方向

      2. 有两个公式

        V是流速，A对应的是投影面积。

      3. 计算系缆力标准值不应大于缆绳的破断力，也不应低于规范规定的下限值，若低于，则取下限值

      4. \(\sum F_x,\sum F_y\)应根据可能同时出现的风和水流的情况，不应将两者最大值叠加，一般可按最大风压力计算值和可能同时出现的水流里来叠加。

    3. 船舶挤靠力（1.系泊于码头的船舶手打风、水流、和波浪的共同作用2.船舶离开码头时，在甩尾过程中，船首对码头的挤压）

      1. 连续布置 \[ F_j=K_j\times \sum F_X/L_n \ \ \ (KN/m) \] \(L_n\)是船舶直线段与设施接触长度

      2. 间断布置 \[ F_j'=K_j' \times \sum F_X/n \ \ \ (KN) \] n为与船舶接触的防冲设施的个数

    4. 挤靠力（相较于挤靠力的持续作用，其作用为接触的一瞬间）（高桩码头、墩柱码头） \[ E_0=U=\rho MV^2_n/2 \ \ \ (KN\cdot m) \] 其中，\(\rho\)为有效动能系数，采用0.7~0.8，\(V_n\)为船舶靠岸时的法向速度（x方向）

      波浪引起的撞击力：一般通过模型试验加以确定

  6. 水流力——由建筑物正面的阻力和背面的涡旋力引起。因而在计算时需要乘上一个阻力系数，这个阻力系数也就和桩基的分布等有关，因为这些因素会改变涡旋力。

重力式码头

重力式码头的结构型式及其特点

主要组成部分及其作用

  1. 胸墙和墙身——挡土是属于墙身的。承受并传递外力、构成整体、便于安装码头
  2. 基础
    1. 扩散、减小地基应力（把接触面积加大了），降低码头沉降（因着减小应力而有的）
    2. 有利于保护地基不受冲刷（应对波浪的淘沙作用）
    3. 便于整平地基，安装墙身
  3. 墙后回填：（主要指抛石棱体（浇注成的棱柱状混凝土，主要为了减少土压力（其作用力向着偏离墙身的方向），同时减小水土流失），倒滤层（一定级配的小石子，需要用布工袋包起来，使得码头地面的下渗水流流失减小））减小土压力，减小水土流失
  4. 码头设施

码头的结构型式

  1. 按墙身结构型式分：方块码头、沉箱码头、扶壁码头、大直径圆筒码头、格形钢板桩码头
  2. 按墙身施工方法分：干地现场浇筑和水下安装的预制结构

方块码头

按断面形式

  1. 阶梯形——断面和底宽较大，稳定性好，基地应力小；但是，材料用量大，横断面方向整体性差，且基底应力分布不均匀
  2. 衡重式
  3. 卸荷板式

按块体形式

  1. 实心方块
  2. 空心方块
  3. 异形方块

沉箱码头

适用：当地有沉箱预制场（有较高的对预制场的要求）或工程量较大、工期短的大型码头

缺点：钢材用量大，耐久性不如方块结构，且需专门的预制下水设备（一般的方法还没把它运下水，需要特制的浮运设备）。

优点：整体性好，抗震能力强，施工速度快，水下工作量少，造价低

  1. 矩形沉箱
    1. 对称式：最常用
    2. 非对称式：节省钢筋混凝土，但制作麻烦，浮游稳定性差
    3. 透空式：对无掩护的港口，效能效果较好
  2. 圆形沉箱——多用于墩式码头
    1. 受力条件好：浮游时产生的是径向的水压力，最终产生环壁的拉应力
    2. 按构造配筋，用钢量少
    3. 空腔内不设隔板，因而重量减小，且空间大，施工方便
    4. 环形箱壁对水流的阻力小

扶壁码头

优点：结构简单、施工速度快、节省材料、造价低

缺点：整体性差、耐久性差

适用：有起重运输设备，有预制能力或有干地施工条件

大直径圆筒码头

特点：

  1. 钢筋混凝土用量少
  2. 对地基适应能力强，可不做抛石基床
  3. 构造简单，较受业主欢迎
  4. 圆通内填料可就地取材

适用条件：地质条件较好的深水码头，或地基表面有不厚但不薄的软土层的情况。

圆筒的制造方法：

  1. 整体预制
  2. 沿高度分成几段
  3. 将圆筒在平面上分成若干片，然后拼装
  4. 现场浇注

格形钢板桩码头

特点：较新型，对地基适应能力强（具有部分板桩码头的属性），施工速度快，占用场地小（减小强的厚度），施工期具有较大的抗风浪能力（可以在有水的情况下进行施工）

重力式码头的构造

在码头设计中，首先要根据当地的自然条件，施工条件，建筑物的使用要求，拟定各种构造措施，即进行构造设计，然后再进行强度和稳定性验算
